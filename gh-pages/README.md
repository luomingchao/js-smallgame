# AI指挥官：智能战线 🤖⚔️

## 🎮 游戏简介

欢迎来到《AI指挥官：智能战线》！这是一款创新的实时策略（RTS）游戏，融合了先进的人工智能技术，为您带来前所未有的游戏体验。

## 🌟 游戏特色

- **🧠 三重AI系统**
  - 雅典娜AI：智能副官，实时提供战术建议
  - 阿瑞斯AI：强大的敌方对手，挑战您的策略
  - 赫尔墨斯AI：动态事件生成器，增加游戏变数

- **⚡ 实时策略战斗**
  - 单位控制：士兵、坦克、工人
  - 资源管理：金币和能量
  - 基地攻防：摧毁敌方基地获得胜利

- **💬 智能对话系统**
  - 与AI副官对话获取战术建议
  - 个性化游戏体验
  - 实时战术分析

- **🎯 渐进式关卡系统**
  - 5个关卡，难度递进
  - 新手教学模式（关卡1）
  - 智能AI难度调整
  - 关卡进度自动保存

- **🔧 用户体验优化**
  - 修复"开始战斗"按钮无反应问题
  - 游戏规则弹窗滚动条支持
  - 响应式设计，适配各种屏幕
  - 详细的错误调试信息

- **⚔️ 战斗系统修复**
  - 修复单位无法攻击敌方单位的问题
  - 添加基地防御机制
  - 实现自动目标分配系统
  - AI独立资源系统（AI和玩家资源分离）
  - 智能AI战术调整

- **⚖️ 策略性战斗系统**
  - 单位克制关系：士兵→坦克→工人→士兵 (石头剪刀布)
  - 克制攻击造成1.5倍伤害，被克制承受0.7倍伤害
  - 基地附近防御加成(20%)
  - 资源点可视化采集范围
  - 玩家基础资源收入(每秒10金币)
  - 工人智能资源采集

## 🎯 游戏玩法

### 基本操作
1. **招募单位**：点击左侧面板的单位按钮招募士兵、坦克或工人
2. **移动单位**：点击地图上的位置移动选中的单位
3. **与AI对话**：在输入框中与雅典娜AI对话
4. **管理资源**：合理分配金币和能量

### 胜利条件
- 摧毁敌方基地
- 或击败所有敌军

### 单位介绍
- **⚔️ 士兵**：基础作战单位，成本100金
- **🚗 坦克**：强大攻击单位，成本300金
- **👷 工人**：采集资源，建造基地，成本50金
