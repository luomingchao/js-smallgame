# 🎮 手动测试指南

## 测试准备
**游戏地址**: http://localhost:8000  
**测试浏览器**: Chrome, Firefox, Safari (推荐 Chrome)  
**测试时长**: 约10-15分钟  

---

## 🎯 测试1: 基地攻击功能

### 操作步骤
1. **打开游戏** - 在浏览器中访问 http://localhost:8000
2. **开始游戏** - 点击"开始战斗"按钮
3. **创建攻击单位** - 点击"士兵"按钮，创建1-2个士兵
4. **攻击基地** - 右键点击敌方基地（地图右侧的红色圆圈）
5. **观察结果** - 观察基地血量条和战斗日志

### 预期结果
- ✅ 士兵自动移动到基地附近并开始攻击
- ✅ 敌方基地血量条从1000开始持续下降
- ✅ 右侧战斗日志显示"对敌方基地造成XX点伤害"
- ✅ 基地被摧毁时显示胜利/失败信息

### 测试验证点
- [ ] 基地血量是否正常下降？
- [ ] 战斗日志是否显示攻击信息？
- [ ] 单位是否正确攻击基地而非停留在原地？

## 👷 测试2: 工人采集功能

- ✅ 多个工人的采集速度明显快于单个工人

### 测试验证点
- [ ] 工人是否自动移动到资源点？
- [ ] 金币数量是否持续增长？
- [ ] 多个工人采集速度是否更快？

## ⚡ 测试3: 能量系统功能

### 操作步骤
1. **观察能量条** - 查看左上角能量显示（⚡）
2. **等待增长** - 观察能量条是否从500开始持续增长
3. **积累能量** - 等待约10-20秒，让能量积累到20以上
4. **创建坦克** - 点击"坦克"按钮（需要300金币+20能量）
5. **验证消耗** - 检查能量是否减少了20点

### 预期结果
- ✅ 能量条持续增长（每秒+5）
- ✅ 坦克创建成功，能量减少20点
- ✅ 新坦克单位出现在玩家基地附近
- ✅ 左侧资源显示正确更新

### 测试验证点
- [ ] 能量条是否持续增长？
- [ ] 创建坦克是否消耗20能量？
- [ ] 能量不足时坦克创建是否被阻止？

## 🚨 异常情况处理

### 如果基地不掉血
1. 检查是否正确点击了敌方基地（红色圆圈）
2. 确认单位已经创建并可以选择
3. 验证战斗日志是否有攻击信息
4. 重启游戏重新测试

### 如果工人不采集
1. 确认工人已经创建
2. 检查地图上是否有资源点显示
3. 观察工人是否有移动行为
4. 等待更长时间观察效果

### 如果能量不增长
1. 检查能量条是否从500开始
2. 等待足够时间（至少30秒）
3. 确认游戏处于运行状态（未暂停）
4. 刷新页面重新开始

### 如果提示信息不准确
1. 尝试创建不同类型的单位
2. 检查缺少的具体资源类型
3. 等待资源恢复后重试
4. 确认提示信息与实际需求匹配
